Accessibility Guide

Learn how to use the QDT Security Dashboard with accessibility features and keyboard shortcuts.

Keyboard Navigation

Basic Navigation

  • Use Tab to navigate between interactive elements
  • Use Shift + Tab to navigate backwards
  • Use Enter or Space to activate buttons and links
  • Use Esc to close modals and panels

Keyboard Shortcuts

  • Ctrl/Cmd + R - Refresh dashboard
  • Ctrl/Cmd + H - Toggle high contrast mode
  • Ctrl/Cmd + M - Toggle reduced motion
  • Ctrl/Cmd + + - Increase font size
  • Ctrl/Cmd + - - Decrease font size
  • Ctrl/Cmd + 0 - Reset font size

Screen Reader Support

Navigation

  • Use the "Skip to main content" link at the top of the page
  • All interactive elements have descriptive labels
  • Status changes and updates are announced
  • Charts and graphs have text alternatives

Announcements

  • Live updates are announced
  • Error messages are announced
  • Status changes are announced
  • Navigation changes are announced

Visual Accessibility

High Contrast Mode

  • Toggle with Ctrl/Cmd + H
  • Enhanced contrast for better readability
  • Clear focus indicators
  • Distinct status indicators

Font Size Controls

  • Increase: Ctrl/Cmd + +
  • Decrease: Ctrl/Cmd + -
  • Reset: Ctrl/Cmd + 0
  • Three size options available

Reduced Motion

Features

  • Toggle with Ctrl/Cmd + M
  • Disables animations
  • Reduces motion effects
  • Improves focus stability

Benefits

  • Reduces motion sickness
  • Improves focus for users with attention disorders
  • Better performance on low-end devices
  • Reduced visual distractions

Best Practices

General Tips

  • Use keyboard navigation when possible
  • Enable high contrast mode in bright environments
  • Adjust font size for comfortable reading
  • Enable reduced motion if animations are distracting
